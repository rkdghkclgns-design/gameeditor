# AETHELGARD 실행 가이드

## 🚀 빠른 시작

### 1. 시스템 실행하기

프로젝트 폴더에서 아래 파일을 **더블 클릭**하세요:

```
start-local.bat
```

또는 터미널에서:

```bash
cd "C:\Users\user\Desktop\유치훈\AI\안티그래피티\GDD 에디터\project-aethelgard"
.\start-local.bat
```

### 2. 실행 확인

2개의 터미널 창이 자동으로 열립니다:

- **AETHELGARD SERVER**: 백엔드 서버 (포트 4000)
- **AETHELGARD CLIENT**: 프론트엔드 (포트 3000)

서버가 완전히 시작되면 (약 5-10초):
- 브라우저가 자동으로 `http://localhost:3000` 열림
- 또는 수동으로 브라우저에서 위 주소 입력

### 3. 시스템 종료

프로젝트 폴더에서:

```
stop.bat
```

또는 각 터미널 창에서 `Ctrl + C` 입력

## 📖 사용 방법

### 홈 화면 (http://localhost:3000)

3가지 작업 모드 선택:

1. **역기획 (MIRROR)**
   - APK 파일 업로드
   - 자동 분석 및 기획서 생성
   - 리소스 추출

2. **AI 창작 (GENESIS)** ⭐ 추천
   - 간단한 프롬프트 입력
   - AI가 전체 기획서 자동 생성
   - 예시: "사이버펑크 세계관의 고양이 RPG"

3. **직접 작성 (MANUAL)**
   - 블록 에디터로 자유롭게 작성
   - 텍스트, 이미지, 오디오, 테이블, 다이어그램 추가
   - AI 코파일럿이 실시간 도움

### 관제탑 (http://localhost:3000/control)

- 모든 프로젝트 관리
- 진행 상황 모니터링
- 시스템 상태 확인
- 새 프로젝트 생성

---

## 🎮 주요 기능

### 블록 에디터

- **텍스트**: 일반 텍스트 입력
- **이미지 (NanoBanana)**: AI 이미지 생성 또는 업로드
- **오디오 (Google)**: AI 음악/효과음 생성
- **테이블**: 데이터베이스 스키마 정의
- **다이어그램**: Mermaid 순서도/ER 다이어그램

### AI 기능

- **AI 코파일럿**: 섹션별 제안 (💡 아이콘 클릭)
- **이미지 생성**: 프롬프트 → 컨셉 아트
- **음악 생성**: 상황 설명 → BGM/SFX

## ⚙️ 문제 해결

### 포트 충돌 (EADDRINUSE)

1. `stop.bat` 실행
2. 다시 `start-local.bat` 실행

### 서버가 시작되지 않음

```bash
cd server
npm install
npm start
```

### 클라이언트가 시작되지 않음

```bash
cd client
npm install
npm run dev
```

### 빈 화면/CSS 안 보임

브라우저 캐시 삭제 후 새로고침:
- Chrome: `Ctrl + Shift + R`
- Edge: `Ctrl + F5`
